ECUACIONES DE SEGUNDO GRADO

Las ecuaciones de segundo grado son de la forma ax^2 + bx + c = 0; donde a, b y c son números reales (que no son cero); donde x se llama variable o incógnita; a y b se llaman coeficientes de las incógnitas y c recibe el nombre de término independiente.

Una ecuación de segundo grado tiene como conjunto solución a dos elementos que se le puede representar como x1 y x2.
